 ## 2025年3-6月中美贸易战宏观环境与市场影响分析

### 摘要

本报告分析了2025年3月1日至6月30日期间中美贸易战关税政策的演变路径及其对A股和港股市场的综合影响。贸易战期间，美国对华关税从20%逐步升至145%，中国采取对等反制措施并叠加资源管制，形成双向冲击。中国央行在6月通过降息、MLF加量和逆回购操作释放流动性，同时推出股票回购增持再贷款，有效稳定了市场情绪。**贸易战对不同行业影响呈现显著分化**，半导体存储芯片价格暴涨，新能源车企业通过技术升级和市场多元化维持增长，稀土板块受益于出口管制政策而表现强势。未来3-6个月，中美贸易关系将进入政策博弈期，关税政策可能呈现"战略妥协"特征，美联储降息预期变化和供应链重构将成为影响A股和港股的关键变量。

### I. 即时冲击层（近1周）

在2025年6月23日至6月30日期间，中美贸易战关税政策进入相对稳定期，但市场仍存在以下即时冲击：

**关键事件概述**：
1. 6月23日，中国央行货币政策委员会召开第二季度例会，提出"加大货币政策调控强度"和"用好证券、基金、保险公司互换便利和股票回购增持再贷款"，释放稳定资本市场信号  。
2. 6月25日，中国人民银行开展3000亿元MLF操作，净投放1180亿元，连续第四个月超额续作  。
3. 6月26日，中证稀土产业指数下跌0.16%，但稀土ETF嘉实近10日"吸金"超2亿元，近1周新增规模同类第一  。

**市场反应**：
- **A股市场**：6月26日上证指数收于3376.20点，小幅下跌0.42%，但市场情绪总体平稳。科技（AI算力、CPO）、消费（商旅文体融合）、有色金属板块表现相对较强  。
- **港股市场**：恒生指数下跌0.05%，恒生科技指数微涨0.51%，显示市场对科技股存在结构性偏好  。
- **跨资产联动**：人民币汇率呈现双向波动特征，6月4日人民币中间价报7.1886元/美元，但月末CNH升值0.48%至7.1634，显示政策对汇率的稳定作用  。期权隐含波动率（如1M ATM波动率）从3.8%降至3.5%附近，反映市场对人民币短期波动预期减弱  。

### II. 政策消化层（近1-4周）

贸易战期间，中美双方通过关税调整和非关税措施展开博弈，政策消化期呈现以下特征：

**政策传导路径**：
1. **美国关税政策**：
   - 2月1日：以芬太尼问题为由，对所有中国输美商品加征10%关税  。
   - 3月3日：关税提升至20%，并扩大至新能源汽车电池、光伏组件，同时启动对中芯国际芯片设备禁运  。
   - 4月2日-12日：实施"双轨制关税"，基准关税10%+附加税34%-125%，总税率升至145%（叠加芬太尼关税）  。
   - 5月14日：日内瓦会谈后，美国取消91%加征关税，保留10%基准关税和20%芬太尼关税，总税率降至30%  。

2. **中国反制措施**：
   - 2月3日-4日：对美煤炭、LNG加征15%关税，对原油等加征10%，并限制钨、碲、铋等关键矿物出口  。
   - 3月4日-5日：对美农产品加征15%-10%关税，限制镓、锗等半导体材料出口，冻结特斯拉上海工厂扩建审批  。
   - 4月4日-12日：对美所有商品加征34%-125%对等关税，并将6家美国企业列入不可靠实体清单  。
   - 5月14日：日内瓦会谈后，中国将关税降至10%，暂停部分非关税反制措施  。

**行业板块表现**：
1. **半导体行业**：
   - 申万半导体指数在6月上涨3.04%，但细分板块分化明显：存储芯片价格暴涨（如DDR4颗粒单月涨幅超100%），而设计板块（数字/模拟芯片）仍承压  。
   - 中国芯片出口量同比增长20.6%，金额增长20%，显示全球需求韧性  。

2. **新能源板块**：
   - 港股蔚来（09868.HK）第二季度净亏损收窄，毛利率10.3%  ；小鹏汽车（09868.HK）第二季度营收同比增长125%，毛利率17.3%创新高  。
   - 中国对美出口劳动密集型产品（家具、玩具）在6月边际改善，但高技术产品（如集成电路）仍受关税和实体清单限制，出口增速放缓  。

3. **稀土板块**：
   - 中证稀土产业指数近1年净值上涨40.85%，处于历史低位的估值水平（PE-TTM仅30.03倍，处于近1年9.41%分位）对资金形成吸引力  。
   - 美国稀土生产商MP Materials停止发运稀土精矿，将助推国内稀土价格上涨  。


### III. 结构重塑层（1-6个月）

贸易战对全球供应链和资本配置格局产生深远影响，未来3-6个月将呈现以下结构性变化：

**产业链转移与重构**：
1. **美国"友岸外包"战略**：在墨西哥和印度建设半导体封测产能，预计2026年实现月产能5万片12英寸晶圆  。
2. **中国供应链多元化**：加速与东盟、俄罗斯的贸易合作，2025年中俄贸易额突破2448亿美元，对东盟出口增长12.3%  。
3. **技术脱钩与自主可控**：中国在半导体设备领域加大研发投入，宁德时代上半年研发投入约100.95亿元，同比增长17.48%  。

**增长前景与行业偏好调整**：
1. **科技板块**：半导体存储芯片价格暴涨支撑设备企业，但设计板块仍面临美国技术封锁压力。
2. **新能源板块**：电池企业通过本地化布局（如宁德时代与福特合作）和钠电池技术突围，维持增长动能。
3. **消费板块**：对美农产品加征关税导致4月进口降幅超50%，可能推高国内饲料成本，但消费复苏政策提供支撑。

**全球资本流向变化**：
- **A股**：外资可能因贸易战风险而降低对高关税行业配置，但政策支持的科技和消费板块仍具吸引力。
- **港股**：恒生科技指数纳入零跑汽车（2025年12月生效）等新能源车企，反映市场对科技属性的认可  。
- **汇率与资本流动**：人民币汇率双向波动但月末升值，显示政策对冲有效，但需关注美联储降息预期变化对资本流动的影响。

### 宏观经济环境概览

2025年3-6月，全球及中美经济呈现以下特征：

| 经济指标 | 美国 | 中国 |
|---------|-----|------|
| GDP增长率 | 1.2%（Q1） | 5.3%（Q1） |
| CPI同比 | 3.4%（5月） | 0.2%（5月） |
| PPI同比 | -1.1%（5月） | -2.3%（5月） |
| 失业率 | 3.6%（5月） | 4.7%（5月） |
| 货币政策 | 暂停加息，市场预期12月降息 | 适度宽松，6月降息10BP，MLF加量续作   |

**全球经济增长动能减弱**，贸易壁垒增多，主要经济体经济表现有所分化。美国通胀压力仍存（CPI同比3.4%），但PPI同比转负（-1.1%），显示生产端压力加大  。中国CPI和PPI环比数据双双负增长，PPI同比、环比均继续下行，在没有通胀掣肘的背景下，维持宽松的货币政策存在合理性  。

**中美贸易摩擦呈现"战略妥协"特征**：美国对锂电池关键矿产（锂、钴、镍、石墨、稀土）豁免关税，但对正极材料（三元、磷酸铁锂）、隔膜等加工环节仍保持高关税限制（如三元材料82.4%税率）  。中国通过出口管制和资源战略，强化了在关键矿产领域的定价权，但也面临国际诉讼风险。

### 中美经济详细分析

**美国经济**：
- **贸易政策目标**：特朗普政府通过关税政策试图减少美国对华贸易逆差，同时保护美国本土制造业和科技产业。
- **政策强度**：关税政策从2月的10%逐步升级至4月的145%，但日内瓦会谈后降至30%，显示政策存在灵活性。
- **市场敏感度**：高关税导致美国消费者购买商品成本大幅增加，可能引发通货膨胀，零售行业协会表示担忧  。

**中国经济**：
- **贸易政策目标**：中国采取对等反制措施，同时通过资源管制和多边合作对冲贸易战影响。
- **政策强度**：中国对美加征关税从2月的10%逐步升至4月的125%，但日内瓦会谈后降至10%，同时保留部分非关税反制措施  。
- **市场敏感度**：贸易战对出口导向型行业（如纺织、家电）形成压力，但对国内消费和科技自主可控领域影响有限。

**特别关注2025年特朗普新政的潜在经济冲击**：
1. **关税政策**：美国对华关税从2月的10%逐步升至4月的145%，但日内瓦会谈后降至30%，显示政策存在灵活性  。
2. **贸易政策**：美国启动对中芯国际芯片设备禁运，限制镓、锗等半导体材料出口，对中国科技产业形成冲击  。
3. **技术管制**：美国将50余家中国高科技企业列入实体清单，包括AI大模型开发、服务器以及超级计算机产业的12家公司  。

### 中央银行政策与市场反应

**中国央行在贸易战期间的货币政策**：

1. **降息操作**：
   - 6月13日：将7天期逆回购(OMO)操作利率和常备借贷便利(SLF)利率下调10个基点  。
   - 6月15日：中期借贷便利(MLF)利率也下调10个基点  。
   - **政策意图**：释放逆周期调节信号，应对贸易战引发的经济压力（如PPI负增长、房地产低迷）  。

2. **流动性工具加码**：
   - 6月6日和16日分别开展1万亿和4000亿买断式逆回购操作，净投放2000亿，为历史首次单月两次大额操作  。
   - 6月25日MLF净投放1180亿元，连续第四个月加量续作  。
   - **政策意图**：应对半年末流动性压力，稳定市场预期  。

3. **股票市场支持政策**：
   - 5月，央行将证券、基金、保险公司互换便利和股票回购增持再贷款合并为8000亿元额度  。
   - 6月23日例会再次强调"用好股票回购增持再贷款"，探索常态化制度安排  。
   - **政策意图**：维护资本市场稳定，缓解贸易战对股市的冲击  。

**政策效果评估**：
- **流动性改善**：6月MLF和逆回购操作保持银行体系流动性充裕，推动社会融资成本处于历史较低水平  。
- **汇率稳定**：人民币汇率双向浮动，在合理均衡水平上保持基本稳定，期权隐含波动率持续回落，市场情绪整体平稳  。
- **股市提振**：政策释放稳定信号，但港股通资金仍呈现净流出趋势，显示外资对贸易战风险的担忧  。

### 政策解读与分析

**中国关键政策解读**：
1. **稀土出口管制**：4月4日中国对7类中重稀土（镝、铽等）实施出口管制，导致4月出口量下降15.55%，但未在日内瓦会谈中取消  。
2. **股票回购增持再贷款**：5月合并为8000亿元额度，6月例会再次强调"用好股票回购增持再贷款"，探索常态化制度安排  。
3. **中小企业数字化转型试点**：6月14日启动试点，中央财政对试点城市给予定额奖励，重点向通用和专用设备制造、汽车制造等行业倾斜  。

**美国关键政策跟踪**：
1. **关税政策**：美国对华关税从2月的10%逐步升至4月的145%，但日内瓦会谈后降至30%，显示政策存在灵活性  。
2. **技术管制**：美国将50余家中国高科技企业列入实体清单，包括AI大模型开发、服务器以及超级计算机产业的12家公司  。
3. **中期选举因素**：美国参议院试图终止全面关税政策，但众议院可能因共和党中期选举压力而阻挠，显示政策存在政治博弈风险  。

**中美关系关键节点事件分析**：
1. **日内瓦会谈**：5月10-12日，中美在日内瓦举行经贸会谈并发布联合声明，双方同意降低关税税率，美国取消91%的加征关税，中国相应调整反制措施  。
2. **斯德哥尔摩会谈**：8月12日，中美在斯德哥尔摩再次举行会谈，双方同意继续暂停实施24%的关税90天，保留10%的关税  。
3. **伦敦会谈**：6月9日至10日，中美在伦敦举行经贸磋商机制首次会议，聚焦稀土出口、技术管制、关税调整及签证政策等核心议题  。

### 行业偏好变化

贸易战对不同行业的影响呈现显著分化，主要行业板块表现如下：

| 行业板块 | A股表现 | 港股表现 | 政策影响 | 市场预期 |
|---------|---------|---------|---------|---------|
| 半导体 | 存储芯片价格暴涨，设备企业受益，设计板块承压 | 科技股受外资流出影响，但半导体设备企业表现较强 | 美国技术封锁和中国国产替代政策 | 短期分化，长期技术自主可控 |
| 新能源 | 宁德时代净利润增长33.33%，毛利率提升至25.02%   | 蔚来净亏损收窄，小鹏毛利率创新高   | 美国关税政策和中国供应链多元化 | 技术升级和市场多元化支撑增长 |
| 稀土 | 中证稀土产业指数近1年净值上涨40.85%，PE-TTM仅30.03倍   | 稀土ETF嘉实近10日"吸金"超2亿元   | 中国出口管制政策   | 供需紧平衡支撑价格上涨 |

**行业偏好变化逻辑**：
1. **半导体行业**：存储芯片价格暴涨支撑设备企业，但设计板块仍面临美国技术封锁压力，资金流向分化。
2. **新能源行业**：电池企业通过本地化布局和技术升级对冲关税压力，国内政策支持和全球需求增长提供支撑。
3. **稀土行业**：出口管制政策抬高国际价格，但国内需求增长和产业升级提供长期支撑，估值处于历史低位吸引资金流入。
4. **农产品行业**：对美出口受关税影响较大，但国内消费政策和替代市场开发提供缓冲。

### 地缘政治动态及其经济影响

**中美贸易战的地缘政治背景**：
- 特朗普政府上台后，贸易政策转向"美国优先"，以芬太尼问题为由对中国输美商品加征关税  。
- 中国采取对等反制措施，同时通过资源管制和多边合作对冲贸易战影响。
- 中美在日内瓦、斯德哥尔摩和伦敦举行多轮会谈，试图缓和贸易摩擦  。

**经济影响分析**：
1. **出口贸易**：中国对美出口在4月同比下降22.8%，但6月降幅收窄至16.1%，显示政策缓和效应  。
2. **进口贸易**：中国自美进口在4月降幅超过50%，但6月进口增速1.1%，今年首次转正，主要受基数效应与能源价格上涨影响  。
3. **供应链重构**：美国在墨西哥和印度建设半导体封测产能，中国加速与东盟、俄罗斯的贸易合作，形成双向供应链调整  。

**地缘政治风险预警**：
1. **关税政策不确定性**：美国参议院试图终止全面关税政策，但众议院可能因共和党中期选举压力而阻挠，显示政策存在政治博弈风险  。
2. **技术脱钩风险**：美国将50余家中国高科技企业列入实体清单，限制镓、锗等半导体材料出口，对中国科技产业形成长期压力  。
3. **多边贸易摩擦**：巴西、多米尼加等国对中国钢管、镀锡板卷加征反倾销税，反映中国出口面临多边压力  。
