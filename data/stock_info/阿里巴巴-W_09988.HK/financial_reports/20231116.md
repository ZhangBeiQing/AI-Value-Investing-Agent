 ## 阿里巴巴2023年三季度报客观评估：符合预期且展现转型成效

阿里巴巴2023年三季度报(对应2024财年第二季度)整体表现符合市场预期，营收和Non-GAAP净利润均实现稳健增长，但增速较前期有所放缓。**财报数据显示，当季实现营收2247.9亿元人民币，同比增长9%，略超市场预期的2240.96亿元；Non-GAAP净利润401.88亿元人民币，同比增长19%，与市场预期基本一致**  。值得注意的是，虽然整体业绩符合预期，但不同业务板块表现差异明显，核心电商(淘天集团)增长乏力，而国际商业、菜鸟和云智能等业务线则展现出强劲增长势头，表明阿里巴巴的业务转型正在取得成效。

### 一、阿里巴巴2023年三季度报关键财务数据

阿里巴巴2023年三季度报对应其2024财年第二季度(2023年7月1日至2023年9月30日)，财报数据显示：

| 财务指标 | 2023年三季度(2024财年Q2) | 同比增长 | 市场预期 |
|---------|------------------------|---------|---------|
| 营收 | 2247.9亿元 | +9% | 2240.96亿元 |
| Non-GAAP净利润 | 401.88亿元 | +19% | 约400亿元 |
| 经营活动现金流 | 492.31亿元 | +4% | - |
| 淘天集团营收 | 976.54亿元 | +4% | - |
| 国际数字商业收入 | 245.11亿元 | +53% | - |
| 菜鸟营收 | 228.23亿元 | +25% | - |
| 本地生活集团收入 | 155.64亿元 | +16% | - |
| 阿里云收入 | 276.5亿元 | +2% | - |

**营收表现方面**，阿里巴巴当季营收2247.9亿元，同比增长9%，略超市场预期的2240.96亿元  。这一增速较2022年同期的19.38%有明显放缓，但仍保持了稳健增长态势。值得注意的是，阿里巴巴在2024财年第一季度(2023年4-6月)的营收增速为13.91%，第二季度增速有所放缓，反映出当前电商市场环境的挑战。

**净利润表现方面**，阿里巴巴当季Non-GAAP净利润401.88亿元，同比增长19%  。这一增速较2023财年同期的-37.29%有显著改善，表明公司降本增效战略取得成效。不过，GAAP净利润为266.96亿元，而上年同期为净亏损224.67亿元，这一扭亏为盈主要得益于投资减值的减少，而非业务基本面的实质性改善  。

**分业务表现方面**，阿里巴巴各业务板块呈现明显分化：

- **淘天集团**：营收976.54亿元，同比增长4%  。其中，客户管理收入(CMR)686.6亿元，同比增长3%；直营及其他收入238.9亿元，同比增长6%。值得注意的是，淘天集团中国零售商业的直销收入同比下降9%，主要由于公司减少了部分低利润业务  。

- **国际数字商业集团**：收入245.11亿元，同比增长53%  。其中，国际零售业务收入189.78亿元，同比增长73%；国际批发业务收入55.33亿元，同比增长9%。这一业务线的强劲增长主要得益于速卖通、Lazada和Trendyol等平台的出色表现。

- **菜鸟集团**：营收228.23亿元，同比增长25%  。菜鸟的盈利能力显著提升，经调整EBITA实现三位数同比增长，主要得益于跨境物流业务的改善和规模效应的显现。

- **本地生活集团**：收入155.64亿元，同比增长16%  。订单量同比增长近20%，饿了么和高德的表现均有提升，其中高德日活跃用户峰值超过2.8亿，创下历史新高。

- **阿里云智能集团**：收入276.5亿元，同比增长2%  。尽管收入增速放缓，但盈利能力显著改善，经调整EBITA同比增长44%至14.09亿元，主要得益于公司减少了低利润率的项目式合约类收入，专注于公共云和AI相关产品的高质量增长。

### 二、市场预期与实际表现对比分析

根据第三方机构的数据，市场对阿里巴巴2023年三季度报的预期如下：

**营收预期**：市场普遍预期阿里巴巴当季营收约为2240.96亿元，实际营收为2247.9亿元，略超预期  。这一超预期表现主要得益于国际商业和菜鸟等业务线的强劲增长，部分抵消了核心电商增速放缓的影响。

**净利润预期**：市场对阿里巴巴当季Non-GAAP净利润的预期约为400亿元，实际为401.88亿元，基本符合预期  。值得注意的是，这一净利润增速较2022年同期的-37.29%有显著改善，但较2023财年第一季度的66.66%有所放缓。

**分业务预期与实际对比**：

- **淘天集团**：市场预期淘天集团的营收增速约为3-5%，实际为4%，基本符合预期。但淘天集团的中国零售商业直销收入同比下降9%，这一表现低于市场预期，主要由于公司主动调整业务结构，减少低利润业务  。

- **国际数字商业**：市场预期国际数字商业的收入增速约为40-50%，实际为53%，超出预期  。其中，国际零售业务的增速更是高达73%，远超市场预期，主要得益于速卖通在欧洲市场的成功拓展和Trendyol的持续增长  。

- **菜鸟**：市场预期菜鸟的营收增速约为20-25%，实际为25%，符合预期  。菜鸟的盈利能力显著提升，经调整EBITA实现三位数同比增长，这一表现远超市场预期。

- **阿里云**：市场预期阿里云的收入增速约为5-10%，实际为2%，低于预期  。不过，阿里云的经调整EBITA同比增长44%，这一表现远超市场预期，表明公司正通过优化收入结构提升盈利能力。

### 三、与腾讯控股2023年第三季度财报的多维度对比

为客观评估阿里巴巴2023年三季度报的表现，我们将其与腾讯控股同期财报进行对比：

**营收对比**：阿里巴巴当季营收2247.9亿元，同比增长9%；腾讯当季营收1546.25亿元，同比增长10.37%  。两者营收增速接近，但阿里巴巴的营收规模更大，显示出其在电商领域的领先地位。

**净利润对比**：阿里巴巴当季Non-GAAP净利润401.88亿元，同比增长19%；腾讯当季Non-GAAP净利润449.21亿元，同比增长39.27%  。腾讯的净利润增速显著高于阿里巴巴，主要得益于其广告业务的强劲增长和游戏业务的复苏。

**分业务收入对比**：

- **电商 vs 游戏**：阿里巴巴核心电商(淘天集团)收入增长4%，而腾讯游戏收入增长7%  ，其中国际市场游戏收入增长14%  。从增速来看，腾讯游戏业务表现略优于阿里巴巴核心电商，但阿里巴巴的电商规模远大于腾讯的游戏业务。

- **广告业务**：阿里巴巴客户管理收入(CMR)增长3%  ，而腾讯广告收入增长20%  ，其中视频号广告收入环比显著增长  。腾讯的广告业务增速远超阿里巴巴，主要得益于其视频号生态的商业化加速。

- **金融科技与企业服务**：腾讯当季金融科技与企业服务收入增长16%  ，而阿里巴巴的云智能业务收入增长2%  ，但盈利能力提升44%  。从增速来看，腾讯的金融科技与企业服务表现更优，但阿里巴巴的云业务正在通过优化收入结构提升盈利能力。

- **国际业务**：阿里巴巴国际数字商业收入增长53%  ，其中零售业务增长73%  ；腾讯国际市场游戏收入增长14%  。**阿里巴巴的国际业务增速显著高于腾讯，显示出其在全球电商市场的竞争力**。

**现金流对比**：阿里巴巴当季经营活动现金流净额492.31亿元，同比增长4%  ；腾讯当季经营现金流净额650.25亿元，同比增长58.73%  。腾讯的现金流增速远高于阿里巴巴，反映出其业务模式的变现效率更高。

### 四、阿里巴巴财报表现的客观评价

综合以上分析，阿里巴巴2023年三季度报的表现总体符合市场预期，但在不同业务板块间表现分化明显。**从整体营收和净利润来看，阿里巴巴略超营收预期，符合净利润预期，显示出公司在降本增效方面的持续努力和成效**。

**核心电商业务表现平平**：淘天集团的营收增速仅为4%，远低于市场预期的3-5%  ，且中国零售商业直销收入同比下降9%  ，反映出电商市场竞争加剧和用户消费习惯变化带来的挑战。不过，淘天集团的客户管理收入(CMR)增长3%  ，表明广告和佣金收入仍保持韧性。

**高增长业务表现亮眼**：国际数字商业、菜鸟和本地生活等业务线均实现两位数增长，其中国际零售业务增速高达73%  ，菜鸟的经调整EBITA实现三位数同比增长  。这些业务的增长主要得益于公司战略转型和全球化布局的成效，显示出阿里巴巴在核心电商之外的新增长点。

**与腾讯对比的启示**：与腾讯相比，阿里巴巴在国际业务和物流领域表现更优，但广告业务和现金流增速落后于腾讯。这反映出两家公司在业务结构和商业模式上的差异，也提示阿里巴巴在提升广告业务效率和优化现金流方面仍有改进空间。

**未来展望**：阿里巴巴集团CEO吴泳铭在财报电话会议上表示，公司将继续推进"用户为先、生态繁荣、科技驱动"的战略，优化业务结构，提升盈利能力  。随着双11大促的临近和AI技术在各业务线的深入应用，阿里巴巴有望在下一季度实现更稳健的增长。

### 五、结论与投资启示

阿里巴巴2023年三季度报整体表现符合市场预期，营收和Non-GAAP净利润均实现稳健增长，但增速较前期有所放缓。**核心电商业务面临增长压力，而国际商业、菜鸟和本地生活等业务线展现出强劲增长势头，表明阿里巴巴的业务转型正在取得成效**。

从投资角度看，阿里巴巴的财报表现显示出公司正在积极应对市场变化，通过优化业务结构和提升运营效率来增强盈利能力。虽然短期内核心电商业务增长乏力，但国际商业和菜鸟等业务的强劲表现提供了新的增长动力。投资者应关注阿里巴巴在AI技术应用和全球化布局方面的进展，以及核心电商业务的复苏情况。

与腾讯相比，阿里巴巴在国际业务和物流领域表现更优，但广告业务和现金流增速落后于腾讯。这反映出两家公司在业务结构和商业模式上的差异，也提示阿里巴巴在提升广告业务效率和优化现金流方面仍有改进空间。

总体而言，阿里巴巴2023年三季度报的表现符合预期，但增速放缓和业务分化也提醒投资者关注公司面临的挑战和转型风险。
